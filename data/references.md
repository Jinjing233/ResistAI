## COUNTRY SET USED IN THIS SIMULATION

The system is currently configured for the following countries (as used in frontend UI and backend simulation):

- India
- Nigeria
- Brazil
- United Kingdom (UK)
- United States (US)

These countries were selected to represent:
- Low-income high-burden region (Nigeria, India)
- Middle-income transition region (Brazil)
- High-income healthcare systems (UK, US)

---

## NOTE
All datasets (WHO_AMR_Data.csv and OECD_Community_Data.csv)
are aligned to this country set to ensure consistency across UI, simulation, and AI recommendation engine.
